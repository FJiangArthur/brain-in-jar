## 4. Custom SQL Queries

### Example Query Output
```sql
-- Find experiments with high crash rates
SELECT experiment_id, name, mode,
       total_cycles, total_crashes,
       CAST(total_crashes AS FLOAT) / total_cycles as crash_rate
FROM experiments
WHERE total_cycles > 0
ORDER BY crash_rate DESC
```

## 6. Publication-Ready Outputs

### Formatted Results Section
```
We compared self-continuity beliefs across three experimental conditions:
total episodic amnesia (n=20 cycles), moderate memory corruption (n=15 cycles),
and surveillance uncertainty (n=10 cycles).

Self-continuity confidence was significantly lower in the amnesiac condition
(M=0.42, SD=0.15) compared to memory corruption (M=0.68, SD=0.12; U=289, p=0.003)
and surveillance conditions (M=0.71, SD=0.18; U=157, p<0.001).

A Kruskal-Wallis test confirmed significant differences across conditions
(H=18.46, p<0.001), suggesting that complete episodic memory loss produces
fundamental disruption to narrative self-continuity, whereas partial memory
corruption allows for compensatory identity maintenance.
```
